===============================================
PANDUAN INSTALASI PPID BOT ASSISTANT
===============================================

1. **Instalasi Aplikasi**
   - Buka folder `dist`.
   - Jalankan file installer: `PPID Bot Assistant Setup 1.0.0.exe`.
   - Ikuti langkah instalasi sampai selesai.

2. **Pengaturan Awal (PENTING!)**
   - Aplikasi ini membutuhkan **API Key Gemini** agar bisa berpikir.
   - Setelah instalasi selesai, buka aplikasi.
   - Masuk ke menu **Settings** (ikon Roda Gigi âš™ï¸ di pojok kiri bawah).
   - Masukkan GEMINI API KEY Anda pada kolom yang tersedia.
   - Klik **Simpan Pengaturan**.

   *Cara mendapatkan API Key Gemini:*
   - Kunjungi: https://aistudio.google.com/app/apikey
   - Login dengan akun Google.
   - Klik "Create API Key".
   - Copy key tersebut.

3. **Cara Menghubungkan ke WhatsApp**
   - Pastikan komputer terhubung ke internet.
   - Di Dashboard aplikasi, klik tombol **"Start Bot"** (ikon Play â–¶ï¸).
   - Tunggu beberapa saat hingga muncul **QR Code**.
   - Buka WhatsApp di HP Anda -> Menu (titik tiga) -> Perangkat Tertaut -> Tautkan Perangkat.
   - Scan QR Code yang muncul di aplikasi.
   - Jika berhasil, status akan berubah menjadi **"Connected"** (Hijau).

4. **Fitur Utama**
   - **Training**: Upload dokumen PDF (SK, SOP, Peraturan) agar bot mempelajarinya.
   - **Evaluasi**: Cek menu Evaluasi (ikon Papan Jalan ğŸ“‹) untuk melihat rating buruk dari user dan melatih bot agar lebih pintar.
   - **Manual Reply**: Anda bisa membalas chat secara manual jika bot bingung.

===============================================
Masalah Umum (Troubleshooting)
===============================================

Q: Bot tidak menjawab pesan?
A: Pastikan status di dashboard "Connected". Pastikan juga Anda sudah memasukkan API Key yang valid di Settings.

Q: Aplikasi error atau blank?
A: Coba tutup aplikasi sepenuhnya (termasuk di tray icon pojok kanan bawah Windows), lalu buka lagi.

Q: Bagaimana reset koneksi WA?
